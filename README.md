# AI Project: Solving the Wumpus World (HCMUS)

## Introduction
This project demonstrates an intelligent agent solving the Wumpus World problem, a classic environment in Artificial Intelligence. The agent explores a grid-based cave containing hazards such as pits and the Wumpus monster, aiming to find gold safely.

##  Approach
We implement a knowledge-based agent using Python. The agent perceives its environment through signals like breeze (near pits) and stench (near Wumpus). Based on these perceptions, it applies logical inference to update its knowledge base and make safe decisions.

##  Key Features
- Grid environment simulation  
- Logical inference system  
- Safe path planning algorithm  
- Decision-making under uncertainty
